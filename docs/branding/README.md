# Mammoth Design Assets

This directory contains source and export assets for the Mammoth brand system.

## Status

The selected no-nonsense Mammoth mark is visually approved and frozen under
`logo/frozen-no-nonsense/`. That kit includes the immutable generated master,
transparent PNG exports, lockup templates, favicons, avatars, social canvases,
and concise usage standards.

The existing canonical SVG files and the frozen kit's composition templates
embed raster artwork. The final production milestone is to trace the frozen
mark as native path geometry, approve a pixel-overlay comparison, convert the
wordmark to paths, and then promote the frozen kit into the canonical asset
locations.

## Directories

- `logo/` — approved mark exports and source references
- `logo/frozen-no-nonsense/` — frozen next-generation mark and brand kit
- `tokens/` — color and typography tokens
- `diagrams/` — architecture diagrams and icon primitives
- `templates/` — reusable documentation and GitHub templates
- `social/` — social announcement copy
