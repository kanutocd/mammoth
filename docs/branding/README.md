# Mammoth Design Assets

This directory contains source and export assets for the Mammoth brand system.

## Status

The approved Mammoth Candidate A artwork is included as PNG and SVG seed assets.

The SVG files currently embed the approved raster artwork. The final production milestone is to replace those seed SVGs with native path-based vector geometry.

## Directories

- `logo/` — approved mark exports and source references
- `tokens/` — color and typography tokens
- `diagrams/` — architecture diagrams and icon primitives
- `templates/` — reusable documentation and GitHub templates
- `social/` — social announcement copy
