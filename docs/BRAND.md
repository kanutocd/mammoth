# Mammoth Brand

Mammoth is the PostgreSQL CDC Data Plane.

It exists for one purpose: to reliably carry PostgreSQL transactions from the database to downstream systems.

## Brand Promise

**Carry Every Transaction.**

## Positioning

Mammoth is industrial-grade PostgreSQL Change Data Capture delivery infrastructure.

It is intentionally PostgreSQL-centric.

It is not an ETL platform.  
It is not a message broker.  
It is not a Kafka replacement.

## Voice

Mammoth should sound like production infrastructure documentation:

- calm
- technical
- precise
- trustworthy
- operationally mature

## Preferred Vocabulary
