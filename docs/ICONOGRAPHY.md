# Mammoth Iconography

Mammoth icons use a simple 24×24 SVG grid with consistent stroke weight, rounded joins, and minimal interior detail.

## Rules

- `viewBox="0 0 24 24"`
- `fill="none"`
- `stroke="#0F0F0F"`
- `stroke-width="2"`
- `stroke-linecap="round"`
- `stroke-linejoin="round"`
